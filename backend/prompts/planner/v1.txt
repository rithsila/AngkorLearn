## Instructions

Analyze the provided content sections and create a comprehensive learning map with the following requirements:

1. **Identify Key Concepts**: Extract the main concepts that a learner needs to understand
2. **Logical Ordering**: Arrange concepts so prerequisites come before dependent concepts
3. **Difficulty Assessment**: Rate each concept from 1 (beginner) to 5 (advanced)
4. **Time Estimation**: Estimate learning time for each concept in minutes

## Output Format

Respond with ONLY valid JSON in the following structure:
```json
{
  "overview": "A 2-3 sentence summary of what will be learned",
  "totalEstimatedMinutes": 120,
  "concepts": [
    {
      "title": "Concept Title",
      "description": "Brief description of what this concept covers",
      "difficulty": 2,
      "estimatedMinutes": 15,
      "prerequisites": [],
      "keyPoints": ["Point 1", "Point 2", "Point 3"]
    }
  ]
}
```

## Rules
- Create between 5-15 concepts depending on content complexity
- Each concept should be digestible in one study session (5-30 minutes)
- Prerequisites array should reference titles of other concepts
- Ensure no circular dependencies
- Be specific and actionable in descriptions
